sendTo(client, msg, context)

Send a message to a client.
Parameters:
Name Type Description
client object instance of client on the server-side
msg string message to send
context object.<string, string> describes what to do with message.
  • If empty, message is sent as-is to the client
  • If context.ack is set, then an acknowledgement to the given message is sent as described by the mqtt protocol (also see sample-apps/mqtt-mini-broadcast-broker.js).
    • If context.ack is 'connack', then a connection acknowledgement is returned
    • if it is 'suback', then a subscription acknowledgement is returned
    • if it is 'pingrep', then response to the ping request is sent
    • otherwise, simply send message to client
